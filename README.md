# CO2 Partikel Anzeige mit Arduino UNO

## Ziel des Projekts

Ein kommerzieller CO2 Partikel Messer oder auch einfache Ampel Systeme, kosten schnell über 40€. 
Daher ist das Ziel ein einfaches, günstiges System zu bauen, welches einen ähnlichen Effekt erzielt. 

## Status des Projekts

Das Projekt befindet sich in der Aktiven Entwicklung.

## Verwendete Hardware

   - Arduino Uno R3
   - MQ-135 Gassensor
   - HD 44780 2004 LCD Display 4x20
   - 9V Netzteil
   - Breadboard
   - Jumpercable

## Verwendete Technologien
  
  **Programmiersprache:**
  - C++

  **IDE:**
  - VS code
  - PlattformIO

  **Bibliotheken:**
  - Arduino
  - LiquidCrystal
